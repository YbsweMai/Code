## 数组
```js
var fruits = ['apple','banana']
fruits.push['orange']  // 在数字最后增加一个元素
fruits.pop(); // 删除数组最后一个元素
fruits.shift(); //删除数字第一个元素
fruits.unshift(''); // 在数组开头添加一个元素
fruits.indexOF('apple'); // 返回元素在数组中的位置
fruits.split
```
