## GIT是什么？
目前世界上最先进的分布式版本控制系统

| 命令                 | 作用                   |
| -------------------- | ---------------------- |
| git init             | 初始化仓库             |
| git status           | 查看当前文件状态       |
| git add xxx          | 将文件放入缓存区域     |
| git commit -m '注释' | 将暂存区上传到本地仓库 |
| git log              | 查看提交记录           |
| git reset            | 重置当前目录版本       |
| git checkout         | 从缓存区拿回文件       |
| git reflog           | 查看对本地仓库所有操   |
| git clone            | 从远程仓库拷贝         |
| git pull             | 从远程库拉取最新代码   |
| git push             | 将本地库推向远程库     |
|                      |                        |

远程库与本地库同步

1.首先github创建库

2.git clone（拷贝）到本地库

3.进入本地库编辑文件（内容）依次上传至暂存区和本地仓库

4.git push  （end）
